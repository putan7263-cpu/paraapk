# PlayStation Store · Поисковик цен (v2)

Программа для быстрого поиска игр в PS Store с расчётом цены в рублях по настраиваемому курсу и надбавке.

## Источник данных

- 🇹🇷 Турция: <https://store.playstation.com/en-tr/pages/browse>
- 🇺🇦 Украина: <https://store.playstation.com/ru-ua/pages/browse>

## Возможности

- 🌍 **Регионы**: 🇹🇷 Турция (₺ лиры) / 🇺🇦 Украина (₴ гривны)
- 🎮 **Автодополнение** при вводе названия игры (поиск в реальном времени из PS Store)
- 💰 **Скидки**: показывает текущую цену + зачёркнутую старую цену + % скидки
- 💱 **Гибкий курс рубля**: вводится вручную, автоматически подгружается из ЦБ РФ
- ➕ **Надбавка**: фиксированная сумма в рублях
- 🔄 **Авто-пересчёт** итога при любом изменении курса/надбавки
- 🔗 **Открытие в браузере**: кнопка для перехода на страницу игры в PS Store

## Как работает расчёт

```
Итого в рублях = (цена_региона × курс_рубля) + надбавка
```

**Пример:**
- Игра стоит **150 TL**
- Курс рубля = **3**
- Надбавка = **140 ₽**
- **Итого: 150 × 3 + 140 = 590 ₽**

## Как пользоваться

1. **Выберите регион** (🇹🇷 Турция или 🇺🇦 Украина)
2. **Введите название игры** в поле поиска (от 2 символов)
3. **Выберите игру** из выпадающего списка
4. **При желании измените курс** (по умолчанию — курс ЦБ РФ)
5. **Введите надбавку**, если нужна
6. Видите **итоговую цену в рублях** — она пересчитывается автоматически

## Технические детали

- **Поиск**: HTTP-запросы к PS Store с парсингом встроенного `__NEXT_DATA__` JSON
- **URL поиска**: `https://store.playstation.com/<en-tr|ru-ua>/search/<запрос>`
- **Курсы валют**: загружаются с сайта ЦБ РФ при запуске (XML API)
- **Дебаунс**: запросы к PS Store не чаще 1 раз в 0.4 сек при вводе
- **Парсер цен**: понимает турецкий (`1.299,00 TL`), английский (`1,299.00`) и украинский (`1 199 ₴`) форматы

## Если что-то не работает

- **Пустой список подсказок** → проверьте интернет; PS Store иногда блокирует запросы по IP — попробуйте через VPN
- **Курс не загрузился** → используется фолбэк (TRY=2.65, UAH=2.20), можете ввести вручную
- **Цена "—"** → игра бесплатная, либо PS Store не предоставил цену
